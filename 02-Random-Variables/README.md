# Random Variables

Understanding random variables and probability distributions for agricultural data modeling.

## Overview

Random variables provide a mathematical way to work with uncertain outcomes. Learn to model agricultural phenomena like crop yields, rainfall amounts, and pest counts as random variables.

## Learning Objectives

- Understand discrete and continuous random variables
- Work with probability mass functions (PMF) and probability density functions (PDF)
- Calculate expected values and variances
- Apply random variables to agricultural modeling
