# Common Probability Distributions

Master the probability distributions most commonly used in agricultural ML and statistics.

## Overview

Learn the distributions that model real agricultural phenomena: Normal for continuous measurements, Binomial for success/failure outcomes, Poisson for rare events, and more.

## Learning Objectives

- Understand and use the Normal (Gaussian) distribution
- Apply Binomial distribution to binary outcomes
- Use Poisson distribution for count data
- Recognize which distribution fits agricultural data
- Understand the Central Limit Theorem
