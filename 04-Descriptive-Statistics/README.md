# Descriptive Statistics

Learn to summarize and visualize agricultural data effectively.

## Overview

Descriptive statistics provide tools to understand data at a glance. Learn to calculate and interpret measures of central tendency, spread, and shape for agricultural datasets.

## Learning Objectives

- Calculate mean, median, and mode for agricultural data
- Understand variance, standard deviation, and range
- Create effective data visualizations
- Identify outliers and unusual patterns
- Summarize distributions effectively
