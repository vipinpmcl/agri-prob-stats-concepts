# Statistical Inference

Learn to make inferences about populations from sample data.

## Overview

Statistical inference allows us to draw conclusions about entire populations (all farms, all crops) from limited samples. Essential for agricultural research and decision-making.

## Learning Objectives

- Understand sampling distributions
- Perform point estimation (MLE, Method of Moments)
- Construct and interpret confidence intervals
- Apply the Central Limit Theorem
- Make inferences about agricultural populations
