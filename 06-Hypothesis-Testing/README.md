# Hypothesis Testing

Learn to test claims and compare groups using statistical methods.

## Overview

Hypothesis testing provides a framework for making decisions based on data. Essential for agricultural research: Does this fertilizer work better? Are yields different across regions?

## Learning Objectives

- Formulate null and alternative hypotheses
- Understand p-values and significance levels
- Perform t-tests, chi-square tests, and ANOVA
- Interpret test results correctly
- Avoid common statistical mistakes
- Compare agricultural treatments
