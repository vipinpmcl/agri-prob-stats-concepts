# Time Series Basics

Analyze sequential data and understand temporal patterns in agriculture.

## Overview

Time series analysis is critical for agriculture: weather patterns, seasonal trends, crop growth over time. Learn to decompose, analyze, and forecast time-dependent agricultural data.

## Learning Objectives

- Understand time series components (trend, seasonality, noise)
- Calculate and interpret autocorrelation
- Decompose time series data
- Test for stationarity
- Perform basic forecasting
- Apply to agricultural temporal data
